# Todo Login App

# 🔐 Login • 📝 Manage Tasks • 🚀 Stay Productive

A beautifully designed React Todo Application with Signup/Login Authentication that helps users securely access and manage their daily activities.

# ✨ Overview

Todo Login App is a beginner-friendly full frontend project built using React.

It combines:

*  User Authentication
*  Page Navigation
*  Task Management UI
*  Beautiful Responsive Design

Users can sign up, log in, and access their personal Todo dashboard after successful authentication.

# How It Works

# Signup Flow

1. User enters username/password
2. Account gets stored in state
3. Redirects back to login page

# Login Flow

1. User enters credentials
2. App validates user data
3. If correct → opens Todo Dashboard
4. If wrong → shows warning message

# Todo Dashboard

After login user sees:

* Welcome header 👋
* Information cards 📊
* Todo management section 📝
